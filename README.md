1-User Class kullanarak kullanıcılarımı oluşturdum.
2- Kullanıcılarım postlarını oluşturabilmesi için createPost() metodu oluşturdum.
3- Kullanıcılarımın oluşturduğu postu göstermek için showPosts() metodu oluşturdum
4-Kullanıcılarım birbirlerinin postlarına yorum yapabilmesi için addCommentToPost() metodu oluşturdum
5-Kullanıcılarım birbirlerinin postunu beğenebilmesi için addToPostFavorites() metodu oluşturdum
6-kullanıcılarım birbirlerini takip etmesi için follow() metodu oluşturdum.
7- Kullanıcılarım birbirlerinin  hesapları ile ilgili bilgilerini akışta görebilmesi showFeed() metodu oluşturdum.

Devamını getirmiş olduğum projede kullanıcılar oluşturdukları postu görüntüleyebilecekler.Aynı zamanda birbirilerini takip edip postlarını beğenip yorum da yazabilecekler. Ve hesaplarını akışta görüntüleyebilecekler.
